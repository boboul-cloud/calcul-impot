### 2. Donnees traitees
L'application fonctionne principalement en local sur l'appareil.

Donnees potentiellement traitees selon les fonctions utilisees:
- Informations fiscales saisies manuellement par l'utilisateur
- Textes extraits de documents importes (OCR)
- Cle API OpenAI saisie par l'utilisateur (stockee localement sur l'appareil)

### 3. Finalites
Ces donnees sont utilisees uniquement pour:
- effectuer des simulations fiscales dans l'application
- proposer des suggestions de cases fiscales
- permettre l'export d'un recapitulatif PDF

### 4. Partage avec des tiers
Par defaut, les calculs sont realises localement.

Si l'utilisateur active les fonctions IA (en renseignant une cle API OpenAI), certains contenus textuels peuvent etre envoyes au service OpenAI pour produire des suggestions ou parser du texte OCR.

Aucune vente de donnees personnelles n'est realisee.

### 5. Conservation
- Les donnees de simulation restent localement sur l'appareil, sauf action explicite de partage par l'utilisateur.
- La cle API OpenAI est conservee localement (UserDefaults) jusqu'a suppression par l'utilisateur.

### 6. Droits des utilisateurs
Vous pouvez:
- supprimer les donnees en effacant l'app
- retirer votre cle API OpenAI dans l'app
- ne pas utiliser les fonctions IA

Pour toute question: [A_COMPLETER]

### 7. Securite
Nous appliquons des mesures raisonnables pour limiter l'acces non autorise aux donnees locales. Aucune methode de securite n'est infaillible.

### 8. Modifications
Cette politique peut etre mise a jour. La date de mise a jour sera modifiee en consequence.
