# Publication App Store - Pack de documents

Ce dossier contient les documents prêts à copier/coller pour la mise en ligne de l'app.

## Fichiers
- APP_STORE_CONNECT_METADATA_FR.md : textes de fiche App Store (FR)
- APP_REVIEW_NOTES.md : notes pour l'équipe de review Apple
- APP_PRIVACY_REPONSES.md : réponses App Privacy (guidage pas à pas)
- PRIVACY_POLICY_FR_EN.md : politique de confidentialité bilingue (Markdown)
- privacy-policy.html : politique de confidentialité bilingue (HTML publiable)

## Ordre recommandé
1. Publier privacy-policy.html sur une URL publique (GitHub Pages ou autre).
2. Copier les textes de APP_STORE_CONNECT_METADATA_FR.md dans App Store Connect.
3. Reporter les réponses de APP_PRIVACY_REPONSES.md dans la section App Privacy.
4. Coller APP_REVIEW_NOTES.md dans App Review Information.
5. Faire un upload TestFlight puis soumettre la version.
