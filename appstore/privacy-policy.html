  • Proposer des suggestions de cases fiscales
  • Permettre l'export d'un recapitulatif PDF

  • La cle API OpenAI est conservee localement (UserDefaults) jusqu'a suppression par l'utilisateur.

6. Droits des utilisateurs

  • Supprimer les donnees en effacant l'app
  • Retirer la cle API OpenAI dans l'app
  • Ne pas utiliser les fonctions IA

The app primarily runs locally on the device.

  • Run tax simulations in the app
  • Suggest tax form boxes

  • The OpenAI API key is stored locally (UserDefaults) until removed by the user.

  • Avoid using AI features
